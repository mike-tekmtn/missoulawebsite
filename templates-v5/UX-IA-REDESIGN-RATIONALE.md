# Templates v5 UX / IA Redesign Rationale

## Scope Completed
- Audited all profession templates in `templates-v5`.
- Expanded each profession to a minimum 4-page architecture:
  - `index.html` (Home)
  - `services.html`
  - `about.html`
  - `contact.html`
- Added consistent top navigation and contextual footer links on every page.

## IA Decisions
1. **Home = fast orientation + primary CTA**
   - Each homepage now leads with a clear value proposition and immediate conversion action.
2. **Services = decision support**
   - Service categories and benefit summaries help users self-segment quickly.
3. **About = trust layer**
   - Credibility, process, and team narrative reduce hesitation before contact.
4. **Contact = low-friction conversion**
   - Dedicated conversion page with clear intent and simplified request flow.

## UX Improvements Applied
- **Clearer navigation patterns:** persistent 4-item nav across all templates/pages.
- **Stronger section hierarchy:** headline → supporting copy → proof blocks → CTA.
- **Conversion-first CTA placement:** above-the-fold primary CTA plus repeated CTA near page end.
- **Richer content density:** expanded service blocks, trust signals, process/context sections.
- **Useful footers:** quick links to high-intent pages (services, forms, scheduling, phone).

## Profession-Specific Layout Strategy
- **Roofing Contractor:** utility dashboard aesthetic, urgency cues, dispatch-focused flow.
- **Dental Clinic:** calm/reassuring clinical style with comfort + insurance trust cues.
- **Law Firm:** editorial/professional narrative emphasizing confidentiality and clarity.
- **HVAC Service:** map/coverage-forward structure with fast scheduling path.
- **Fitness Coach:** funnel-style layout with outcome framing and application CTA.
- **Home Cleaning:** asymmetric utility cards with quote-first interactions.

## Outcome
`templates-v5` now has distinct, profession-appropriate multi-page UX systems with stronger conversion pathways and improved information architecture.
