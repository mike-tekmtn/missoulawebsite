# Missoula Website Co

Marketing site for missoulawebsite.com focused on one-day website builds for local Missoula businesses.

## Local run

```bash
cd site
node server.js
# open http://127.0.0.1:3080
```

## Current scope

- Dark theme homepage with intake form
- Complexity scoring + feasibility summary on submit
- Telegram alert to owner on each intake
- Services and contact pages

## Next planned scope

- SEO service package page
- Hosting operations page
- Case studies/testimonials section
