# missoulawebsite.com — Strategy + Copy Draft (v1)

## 1) Positioning + Offer Structure

### Positioning statement
**We build and host fast, trustworthy small-business websites for Missoula-area service companies — with local support and zero tech stress.**

### Ideal customers
- Solo operators and small teams (1–20 people)
- Local services: contractors, plumbers, electricians, HVAC, landscapers, cleaners, med spas, photographers, legal/accounting, etc.
- Businesses that rely on calls, quote requests, and local reputation

### Core value props
- **Local-first:** Built for Missoula buyers, not generic templates
- **Done-for-you:** Design, copy, launch, hosting, edits, support
- **Lead-focused:** Clear calls-to-action, mobile-first, speed, local SEO basics
- **No ghosting:** Real human support, simple monthly plans

### Offer ladder
1. **Launch Site (Starter)**
   - 1–5 pages, mobile-first, contact form, Google map embed, analytics setup
   - Best for new or very small businesses
2. **Growth Site (Most Popular)**
   - 8–12 pages, service pages, testimonials, FAQ, blog-ready CMS
   - Local SEO foundations and conversion-focused copy
3. **Pro + Care Plan**
   - Everything in Growth + priority edits, monthly updates, uptime/security monitoring, backups, performance tuning

### Pricing model recommendation
- **One-time build fee + monthly hosting/care fee**
- Example framing:
  - Starter build: from **$1,500** + **$79/mo** hosting/care
  - Growth build: from **$3,000** + **$129/mo** hosting/care
  - Pro build: from **$5,000** + **$199/mo** care
- Optional: “No large upfront” split-pay offer for cashflow-sensitive businesses

## 3) Visual Direction

### Brand vibe
- **Modern Montana local-business confidence**: clean, grounded, friendly

### Suggested palette
- Pine: `#1F4D3A` (primary)
- Glacier blue: `#3B82F6` (accent CTA)
- Warm sand: `#F5F2EA` (background alt)
- Charcoal: `#1F2937` (text)
- White: `#FFFFFF`

### Typography
- Headings: **Manrope** or **Sora**
- Body: **Inter**
- High contrast, readable sizes (16–18px body)

### UI style
- Spacious sections, clear hierarchy
- Rounded cards, subtle shadows
- Real local imagery (Missoula landmarks + local biz contexts)
- CTA buttons with clear verbs: “Get My Quote”, “Book a 15-Minute Call”

## 5) Implementation Plan + First Build Steps

### Build stack recommendation (fast + maintainable)
- **Next.js + Tailwind CSS**
- Deploy on **Vercel** (or managed VPS if preferred)
- Forms: Formspree / Basin / custom API route to email
- Analytics: Plausible or GA4

### 2-week launch plan
**Day 1–2:** Brand direction, sitemap, copy outline, CTA strategy  
**Day 3–5:** Homepage + core pages design/build  
**Day 6–8:** Services/Pricing/Contact + SEO metadata + performance polish  
**Day 9–10:** QA, mobile checks, launch, analytics + conversion tracking

### Immediate next actions for Mike
1. Confirm package names and pricing anchors
2. Gather 3–5 local proof points (clients, testimonials, logos, outcomes)
3. Pick booking workflow (Calendly or simple form-first)
